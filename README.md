  # Installation
  ```terminal
npm init
``` 
```terminal
npm install express sequelize mysql2
``` 
  ```terminal
mysql -u root -p
``` 
enter your mysql password
  ```terminal
source db/schema.sql
use ecommerce_db
quit
``` 
  ```terminal
npm run seed
node server
``` 
I utlized postman in order to view and adjust the information. You may utilize postman, insomnia or any other api site.
